# Travel List Application 
![App Screenshot](https://github.com/Igor-Puscikowski/Travel_List/blob/master/travel%20list.png)

# Description:

Travel List is a cutting-edge application developed using React, designed to assist travelers in organizing their trips efficiently. Leveraging the power of React's stateful components and props, Travel List offers users a seamless and intuitive experience for planning and managing their travel itineraries.

# Packing list features:

- You can add items to the list,
- specifying their quantity and category (e.g. drinks, clothes, electronics).
- The app has a search option that makes it easy to find the items you need.
- You can mark items as packed to track your packing progress.
- The app displays a percentage indicator of the packed items.
